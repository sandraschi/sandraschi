# 🐦 The Pelican

The bicycling pelican is the mascot of this repo and a small insider joke for anyone who follows AI benchmark culture.

In October 2024, Simon Willison published a [benchmark test](https://simonwillison.net/2024/Oct/21/claude-artifacts/#the-pelican-on-a-bicycle-test) for AI models: **draw a pelican riding a bicycle**. It became an informal community standard for evaluating generative AI visual reasoning — not because it's technically hard, but because it requires understanding anatomy, physics, and absurdity simultaneously.

The pelican in this repo was generated by **Claude Sonnet 4.6** as an SVG artifact, then refined manually. It cycles forever in a loop, which feels appropriate.

The pelican belongs on the main README and not hidden here — but the *why* belongs here, which is why you're reading this.
