# 🤖 Noetix Bumi Technical Profile

**Canonical MCP**: [`bumi-mcp`](file:///d:/dev/repos/bumi-mcp)

The **Noetix Bumi** is a compact, home-friendly humanoid robot that has become a critical focus for the Alsergrund node. While high-end platforms like the Unitree G1 capture the headlines, the Bumi represents the **"iPhone-priced"** inflection point for humanoid robotics.

---

## 🏗️ Hardware Overview

| Specification | Details |
| :--- | :--- |
| **Class** | Education/Consumer Humanoid |
| **Pricing** | ~€1,200 - €2,500 (Variant dependent) |
| **OS Support** | Ubuntu / ROS 2 Friendly |
| **Philosophy** | Accessible, modular, and workbench-stable |

### Why the Bumi Matters
1. **Accessibility**: At the price of a smartphone, it lowers the barrier to Sim2Real research.
2. **Compact Form Factor**: Unlike the 25kg Unitree units, the Bumi is optimized for desktop and indoor apartment environments (like the Alsergrund node).
3. **Ecosystem**: Strong integration with JD.com and Chinese consumer robotics standards ensures a steady flow of parts and community support.

## 🌐 Software & Sim Integration

The [`bumi-mcp`](file:///d:/dev/repos/bumi-mcp) server provides:
- **Hero Specs**: Real-time access to the platform's physical capabilities.
- **Virtual Twin**: Logic for driving a Bumi vbot in **Resonite** via **robotics-mcp** and **unity3d-mcp**.
- **Market Monitoring**: Automated status checks for pre-order windows on JD.com.
